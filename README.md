# 🛒 E-Commerce Website UI

## 📖 About the Project

This project was created as part of the **CodSoft UI/UX Design Internship**.

The objective of this task was to design a modern and visually appealing e-commerce website for collectible toy enthusiasts. The interface focuses on providing an engaging shopping experience through intuitive navigation, organized product categories, and detailed product pages.

## 🎯 Design Objectives

- Modern shopping interface
- Easy product browsing
- Attractive product showcase
- Clear navigation structure
- Responsive layout concept
- User-friendly shopping experience
- Visually engaging product pages
- Consistent design aesthetics

---

## ✨ Key Features

- Homepage with featured collections
- Product category section
- Best seller showcase
- Product details page
- Wishlist functionality
- Search bar
- Shopping cart access
- Customer reviews and ratings
